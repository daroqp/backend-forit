# Backend ForIt 2025

1. Instalar todas las dependencias:
```
npm install
```

2. Copias el archivo .env.template a .env:
```
cp .env.template .env
```

3. Configura las variables de entorno del archivo .env:
```
PORT=3000
```

4. Ejecutar el script dev para usar la app en modo desarrollo:
```
npm run dev
```
